# Password Tools

A Python-based utility for managing and generating secure passwords. This project is designed to simplify password-related tasks, such as generating random passwords, validating password strength, and more.

## Features

- Generate secure random passwords.
- Validate password strength based on customizable rules.
- Easily extendable and configurable.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ostans/password_tools.git
   cd password-tools
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
